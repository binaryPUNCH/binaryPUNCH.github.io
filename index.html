Welcome to my site!

Its main purpose is for me to play with the technologies I find interesting and wish to learn. The server is hosted on an Amazon EC2 instance (vps), running as a free-tier t2.micro instance (Ubuntu-x64 OS).

Brief directory of what to find:

  • tech-flex.com
    The static page you're currently viewing, actually resides at GitHub Pages and not my VPS (awesome for free static sites).

  • blog.tech-flex.com
    My personal blog using the (in)famous Wordpress. All the posts are 5+ years old, so don't judge me :-) I do, however, have plans for more content in the near future (heads up for "Continous-deployment with AppVeyor" & some "Basic JSON.NET use cases").

  • mvc.tech-flex.com
    My ASP.NET Core MVC 6 playground, with a slick deploy strategy. Commits to the master git-branch are automatically built & deployed live. Thanks to AppVeyor for freely enabling this continous-delivery architecture (at the "cost" of staying open-source, but this is totally cool!). For a future blog post, I will go in-depth with the deployment strategy and how it works.

    • Entity Framework Core connected to a MySQL Amazon RDS database
    • Future goals: Add javascript SPA functionality & webpack (likely with Angular2 or Vue.js, maybe React)

  • www.jakupwhansen.dk
    A bonus link to my coding teachers personal site, to whom I owe a lot of inspiration from.

Nginx is used for the blog and Kestrel for the .NET Core site (however Nginx sits in front proxying, since they can't both bind to port 80). Supervisor is used to manage (start/stop/update) .NET Core. Would love to encapsulate the entire project here to a container with autoscaling CloudFormation (loadbalancers, Route3 and everything... a future project).
