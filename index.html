tech-flex.com

Welcome to tech-flex.com

Its main purpose is for me to play with the technologies I find interesting and wish to learn. The server is hosted on an Amazon EC2 instance (vps), running on the free-tier t2.micro instance (Ubuntu-x64 OS).

Brief directory of what to find:

  • tech-flex.com This static page you're currently viewing, built with Jekyll, and actually isn't on the VPS, but resides at GitHub Pages (awesome for a free static site)
  • blog.tech-flex.com my personal blog running on Wordpress, if it was not obvious. All the posts are +5 years old, so don't judge me :-). I do, however, have plans for more content in the near future (Heads up for "Continous-deployment with AppVeyor" & some "basic JSON.NET usecases")
  • mvc.tech-flex.com My ASP.NET Core MVC6 playground, with a slick deploy strategy. Commits to the master git-branch are automatically built & deployed live. Thanks to AppVeyor.com for allowing continous-delivery strategy for free. I will go in depth about this Read more about it in my blog post here.
    • Entity Framework Core connected to a MySQL Amazon RDS database
    • Future goals: Add javascript SPA funcitonality & webpack (likely with Angular2 or Vue.js, maybe React).

Nginx is used for the blog and Kestrel for the MVC6 site (however Nginx sits in front proxying, since they can't both bind to port 80). Supervisor is used to manage (start/stop/update) MVC6. Would love to encapsulate the entire project here to a self-scaling container with CloudFormation (loadbalancers, Route3 and everything... project for another day)

There is value in taking a moment to think about the deep and broad stack that your application sits on. Go thank and support the projects, both open source and not, that your application uses. Revel in the layers of abstraction that others have created and appreciate the ones that you have created. They make all the LEGO pieces you're using just the correct size, and they make snapping them together a lot of fun.
